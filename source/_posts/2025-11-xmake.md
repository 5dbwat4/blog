xmake 是一个跨平台的自动化构建工具，类似于 CMake 和 Make，但更现代化和易用。https://xmake.io/

# 安装

在Windows下直接下载Prebuilt Release即可使用。

1. 在[这里](https://github.com/xmake-io/xmake/releases)下载最新的Release版本`xmake-[version].win64.exe`
2. 直接双击安装，他会自己处理PATH

其它系统&安装方式参考官方文档：[Quick Start | Xmake](https://xmake.io/guide/quick-start.html#installation)。

# 创建项目

```bash
xmake create myproject
```

# 写~~CMakeLists.txt~~xmake.lua

xmake使用`xmake.lua`作为项目的构建脚本。
